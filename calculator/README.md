# 🧮 NeonCalc - PyQt5 Calculator

A modern and responsive desktop calculator built using **Python** and **PyQt5**.
This project focuses on GUI design, user interaction, and basic expression evaluation.

---

## 🚀 Features

* Clean and colorful UI
* Basic arithmetic operations (+, -, ×, ÷)
* Decimal support
* Input validation (prevents invalid operator usage)
* Clear button to reset input
* Responsive button layout using Grid System

---

## 🛠️ Technologies Used

* Python 🐍
* PyQt5 🎨

---

## ▶️ How to Run

1. Install dependencies:

```
pip install PyQt5
```

2. Run the program:

```
python calculator.py
```

---

## ⚠️ Note

* This calculator uses Python's built-in `eval()` function for expression evaluation.
* For production-level applications, a safer parser should be used.
